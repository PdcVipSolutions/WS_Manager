Добавление и удаление файлов

Операции добавления и удаления файлов выполняются с использованием разделов Add File(s) и Manipulate Entity.


Добавление файла

Файлы в группу могут быть добавлены только те, которые имеют расширения, зарегистрированы в свойствах рабочего пространства.
Для добавления файла в группу:
  • Выберите целевую группу в дереве рабочего пространства. Если группирование не производится, то выбирается корень дерева.
  • В разделе File Editing вызовите операцию Add File.
  •  В появившемся диалоге выберите директорию и расширение файла.
           
  • Закройте диалог кнопкой Open или отмените операцию.
  • Если файл был выбран, то в списке файлов рабочего пространства появится добавленный файл.

Групповое добавление файлов

Групповое добавление файлов из выбранной директории выполняется с использованием иконки Add  From Folder.
При этом вызывается диалог для выбора директории. Из выбранной директории в выбранную группу добавляются все файлы, расширения которых зарегистрированы в текущем рабочем пространстве (WorkSpace). Видны же будут только файлы, расширения которых выбраны с помощью фильтра.

Удаление файла

Для удаления файла:
  • Отметьте файл и в панели управления активируйте кнопку Delete File.
  • Подтвердите удаление файла, если операция вызвана не по ошибке.
Выбранный файл может быть также удален клавишей Delete клавиатуры.
Для удаления группы файлов следует отметить последовательно список файлов и использовать либо иконку Delete File, либо кнопку Delete клавиатуры.

Примечание: удаление файлов из папки не производится (в отличие от группы), удаленные файлы лишь помечаются своим (выбранным) цветом.

Проверка существования файлов

Если файл включен в список файлов рабочего пространства, но его физически не существует WSM обнаруживает это при:
  • загрузке данных файла рабочего пространства
  • при выполнении операций над файлами
  • при использовании иконки Check Files
В последнем случае проверяется физическое существование файлов: принадлежащих выбранному узлу (и всем его подузлам).
Текущая версия WSM не производит проверку существования файлов непрерывно.
